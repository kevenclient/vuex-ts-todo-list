## Creating a Vue project with Vuex + Typescript

Create a new project

```bash
vue create vuex-ts-todo-list
```

Presets
```bash

? Please pick a preset:
  Default ([Vue 3] babel, eslint)
  Default ([Vue 2] babel, eslint)
> Manually select features
```
